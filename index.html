    Projects

    Booking system data insights - Python

    Python script was created to quickly provide insights from a booking system dataset of a recycling center in London. Pandas, numpy, matplotlib, seaborn libraries and widgets were utilised. Data cleaning, manipulation applied and main KPIs established.

    Main conclusion:Data has shown that the number of daily bookings against the capacity is relatively low and there are opportunities to reduce operational costs by either reducing the number of staff required.

    View Python script

    Recycling Centre dashboard - Excel

    A waste data flow dashboard was created in Excel for a recycling centre in London to provide environmental and operational KPIs. This dashboard is divided in 3 sections: Household waste Recycling Centre area (where residents dispose of household waste), Transfer Station area (used by businesses and borough kerbside vehicles) and overall (average between both areas).

    Main conclusion: Considerable operational changes needs to be implemented to increase the recycling rate at the Transfer Station area, as it's deflacting the overal site recycling KPI. However, it's important to consider that the Transfer Station area handles 10x more waste and it could limit the efficiency of new waste segregation processes.

    View Excel dashboard

    Worldwide meat production - SQL and Tableau

    In this project, the "worldwide meat production from 1961 to 2021" is analysed using SQL server. Some data cleansing was performed and all insights are shown in a Tableau dashboard.
    Data source: https://ourworldindata.org
